About

測定結果をプロットして表示するツール

主な機能

  • XY直行座標のプロット

  • 極座標プロット

  • ヒートマップ (色による3次元プロット)

対応ファイル形式

CSV

2次元データの場合は

! comment message
x, y1, y2, y3
0,  1,  2,  3
1,  1,  2,  3
2,  1,  2,  3
3,  1,  2,  3

のように

  • 先頭に!をつけてコメント

  • コメントを除いた1行目にデータの種類(軸)

  • 2行目以降に各データ を満たす必要がある.

3次元データの場合は

! comment message
x, y, z, f1, f2, f3, ...
0, 0, 0, 10, 20, 30, ...
1, 0, 0, 10, 20, 30, ...
2, 0, 0, 10, 20, 30, ...

のようにx, y, zが必要になる
